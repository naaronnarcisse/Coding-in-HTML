Tools to build a website

Websites are easy to make. Don't believe me? I started learning how to create them just last week. The reason it has been so easy is because the tools used to make them are easy to understand and use. Tools like HTML, CSS, IDEs, and so on have been incredibly easy to grasp. So easy, that I believe anyone can do it.

HTML

HTML stands for HyperText Markup Language. HTML is all you really need to make a website, but it will look extremely boring. HTML lets the browser know what the content we are writing actually is.

All content must be defined using HTML. Whether it's the heading, the paragraphs, the bold, the italic, the list, or the link, everything must be explicitly defined. By now, you should understand that HTML is all about the content of the webpage.

CSS

CSS stands for Cascading Style Sheets. It revoles around the look of our websites. So, while HTML deals with the content on the page, CSS styles the content and makes it look interesting.

CSS grabs the attention of the user, so that the useful content, brought by HTML, can be consumed. The format for CSS is all about pairs, specifically "property: value;" pairs. Properties are what we want to change and values are what we want to set the property to. CSS is full of these pairs and because of that we can create a website that carries a piece of our identity.
